# A-Quantitative-Comparison-of-Analytical-Models-for-Helical-Tendon-Driven-Continuum-Robots
This repository serves as the digital appendix for the bachelor project "Tendon actuated continuum robots" by Nilas Lorenzen, Peter Fabiansen, and Rasmus Møller (Aarhus University, Spring 2026). The project evaluates the trade-off between modeling fidelity, computational efficiency, and physical accuracy in highly non-linear soft robotic systems.

This repository contains the complete source code for our numerical modeling, physical control, and computer vision tracking, alongside raw and processed experimental data.
